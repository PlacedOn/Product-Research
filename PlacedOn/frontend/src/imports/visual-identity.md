# Visual Identity

PlacedOn should feel like a high-trust hiring product, not a generic AI dashboard.

## Shared Principles

- Candidate surfaces should feel calm, humane, premium, and agency-first.
- Employer surfaces should feel denser, more operational, and decision-oriented.
- Keep one consistent visual system across landing, candidate, interview, and employer pages.
- Use product UI storytelling, not stock-photo marketing.
- Avoid fake traction claims and fake major customer logos unless clearly marked as sample/demo.

## Palette Direction

Prefer:

- Near-black text: `#1F2430`
- Primary action blue already used in the UI: `#3E63F5`
- Trust green: `#10B981`
- Deep green for employer/trust surfaces: `#0A6847`
- Warm off-white/neutrals: `#F8F7F5`, `#F3F2F0`, `#FFFFFF`

Avoid:

- A new purple-heavy palette
- Cyberpunk/neon styling
- Heavy gradients on every section
- Decorative orbs as the main design device

## Component Style

- Use `font-[Manrope,sans-serif]` for headings where current pages do.
- Use soft cards, subtle borders, and restrained shadows.
- Keep rounded cards consistent with the existing dashboard.
- Do not put cards inside cards unless the nested element is a genuine repeated item or modal.
- Mobile layouts must avoid text overlap and horizontal overflow.
